# Wireframes

Screenshots of the running application are in `.EN_Docs/media/`.

## Dashboard — My Documents tab

Shows the main document table with upload sidebar, folder tree, category/tag filters, search bar, and action buttons (download, edit, delete, share).

See: `.EN_Docs/media/dashboard.png`

## Dashboard — Shared with Me tab

Shows documents shared with the current user. Download button only (no edit/delete on shared docs).

See: `.EN_Docs/media/shared.png`

## Edit Document Modal

Inline modal for editing title, category, tags, description. Opened via ✏ button in document row.

See: `.EN_Docs/media/edit-modal.png`

## Share Document Modal

Modal for entering recipient username. Opened via green Share button in document row.

See: `.EN_Docs/media/share-modal.png`

## Folder Sidebar

Collapsible folder tree with ▶/▼ toggles. Parent folders expand to show subfolders. File icons per extension shown for leaf documents.

See: `.EN_Docs/media/folders.png`

## Password Reset Flow

1. Login page → "Forgot your password?" link
2. Email entry form
3. Inbox reset email
4. New password form

See: `.EN_Docs/media/reset-flow.png`

## Dark Mode

Same dashboard with dark colour scheme applied.

See: `.EN_Docs/media/dark-mode.png`
