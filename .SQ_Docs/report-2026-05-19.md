## 3. Problemet

- **Çështje të hasura:**
  - `loadFolder(id, this)` po thirrej mbi një `<span>` brenda div-it folder-item kur dosjet kanë fëmijë — klasa CSS `.active` do të ulej mbi span-in në vend të kontejnerit `.folder-item`, duke prishur theksimin
- **Zgjidhja:**
  - U ndryshua onclick-u në `loadFolder(id, this.closest('.folder-item'))` kështu që klasa aktive gjithmonë aplikohet mbi kontejnerin e saktë pavarësisht cilit element fëmijë i klikua
