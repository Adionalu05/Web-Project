## 3. Issues

- **Issues encountered:**
  - `loadFolder(id, this)` was called on a `<span>` inside the folder-item div when folders have children — the `.active` CSS class would land on the span instead of the `.folder-item` container, breaking the highlight
- **Resolution:**
  - Changed the onclick to `loadFolder(id, this.closest('.folder-item'))` so the active class is always applied to the correct container regardless of which child element was clicked
